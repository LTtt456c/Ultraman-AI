# Ultraman-AI


Ultraman-AI 是一个基于人工智能的奥特曼知识库系统，集成了奥特曼的知识图谱、人物关系图谱，并且支持基于实时查询的问答功能。通过RAG（Retrieval-Augmented Generation）技术，用户可以随时获取奥特曼相关的知识信息。


## 功能


- **奥特曼知识图谱**：构建了奥特曼宇宙中的丰富知识图谱，包含奥特曼系列的怪兽、战士、故事背景等详细信息。
- **人物关系图谱**：展现奥特曼及其他角色之间的关系图谱，帮助用户了解角色间的互动。
- **实时问答**：通过RAG模型，用户可以进行基于实时数据的查询，获得相关问题的答案。


## 技术栈


- **RAG（Retrieval-Augmented Generation）**：用于实时知识查询和生成。
- **Python**：核心开发语言。
- **Graph Database**：用于存储和管理奥特曼的知识图谱。
- **Hugging Face Transformers**：自然语言处理库。
